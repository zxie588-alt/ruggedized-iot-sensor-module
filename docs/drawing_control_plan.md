# Drawing Control Plan

Document status: concept drawing-control evidence for portfolio review.

## Purpose

This note upgrades the drawing package from visual concept sheets to an early inspection-control package. It shows how the enclosure would be moved toward a manufacturing drawing release without claiming supplier-approved production control.

## Added Evidence

- `drawings/iot_sensor_module_v2_inspection_control_sheet.svg`
- `drawings/inspection_characteristics_v1.csv`
- Updated drawing register with DWG-003.

## Control Logic

- Datum A is the lower enclosure base/mounting face.
- Datum B is the gasket groove centreline and sealing-path reference.
- Datum C is the service/cable boss axis for interface alignment checks.
- CTQ characteristics cover enclosure envelope, gasket groove, screw boss position, PCB support height, service port position, strap lug spacing, lid compression rib, battery clearance and cable gland alignment.

## Engineering Meaning

The package demonstrates drawing release thinking: feature control, datum selection, inspection method planning and evidence boundaries. It is suitable as concept portfolio evidence for product design, manufacturing engineering and quality-facing mechanical roles.

## Boundary

This is not a production-controlled drawing release. Final GD&T symbols, tolerance values, material notes, moulding shrinkage allowances, supplier inspection plans and first-article records must be completed after final CAD, material choice, manufacturing method and prototype review.
