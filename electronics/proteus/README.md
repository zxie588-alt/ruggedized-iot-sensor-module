# Proteus Electronics Folder

Current outputs:

- Electronics architecture note for MCU, sensor, battery, regulator, LEDs, debug/service port and radio boundary.
- Concept signal table linking electronics blocks to firmware ownership.
- Portfolio architecture diagram in `images/`.

Planned optional output:

- Real Proteus `.pdsprj` schematic capture and exported schematic image after component selection.

Boundary:

Proteus is used to communicate electronics architecture and support mechanical packaging. This folder is not intended to claim a final PCB manufacturing release or validated circuit simulation yet.
